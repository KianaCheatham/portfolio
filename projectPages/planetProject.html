AR Solar System Model

  • Xcode
  • Swift
  • Blender
  • Reality Converter
  • Figma
  • Procreate

For a personal project, I designed and developed an AR app that allows users to explore planetary models in physical space. The system investigates spatial interaction as a method for understanding astronomical scale and structure.


This project was created in Xcode with Swift and RealityKit, using Blender for 3D models, Procreate for planet textures, and Figma for UI design.
